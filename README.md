# 💻 Personal Portfolio

This repository contains my personal portfolio website, where I showcase my projects, skills, and experience as a full-stack developer.

The portfolio is designed to be interactive, visually engaging, and reflective of my approach to building modern web applications.

## 🌐 Live Demo
👉 https://portfolio-aama.vercel.app/

## 📖 About

This portfolio highlights:

- Real-world projects I’ve developed  
- Full-stack applications and systems  
- Experience in automation and business solutions  
- My technical stack and areas of interest  

It is built with a strong focus on user experience, smooth animations, and a unique pixel-style design inspired by video games.

## ⚙️ Features

- Interactive UI with scroll animations  
- Project showcase section  
- Responsive design  
- Modern and clean layout  
- Fast deployment using Vercel  

## 🛠️ Tech Stack

- Frontend: (React / Next.js / HTML / CSS / JS)
- Styling: (Tailwind / CSS / etc.)
- Deployment: Vercel

The goal of this portfolio is to present my work in a professional and engaging way, making it easy for recruiters, companies, and collaborators to understand what I can build and the value I can bring.
